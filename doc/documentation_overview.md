# LeanTime Documentation Overview

Welcome to the LeanTime architecture and agents documentation!

For clear separation of concerns, all agent/architecture details are divided into two focused documents:

- [Client AGENTS.md](../src/client/AGENTS.md) – All client-side architecture, usage, and conventions
- [Server AGENTS.md](../src/server/AGENTS.md) – All server-side architecture, usage, and conventions

## How to Use This Documentation

- For frontend or UI work, see **Client AGENTS.md**.
- For backend, API, or timer logic, see **Server AGENTS.md**.
- If making changes that affect event contracts or cross-boundary logic, always update both documents.

Each document includes:
- Tech stack and file map
- Architecture diagrams
- Startup/build/testing info
- Checklists for safe edits
